# AoTomate Cockpit

Estrutura inicial do cockpit web para o projeto AoTomate Sistemas.

## Como começar

1. Abra um terminal em `c:\Users\User\OneDrive\Documentos\AOTOMATE\frontend`
2. Rode `npm install`
3. Rode `npm run dev`

## O que está incluído

- Frontend React + Vite + TypeScript
- Layout inicial de dashboard com resumo, cards e painel rápido
- Estrutura pronta para adicionar rotas, APIs e integrações
